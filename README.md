# 🍕 Pizza Sales SQL Analysis

## 📌 Project Overview

This project analyzes a pizza sales dataset using **MySQL** to generate actionable business insights. It demonstrates SQL skills such as data retrieval, joins, aggregate functions, subqueries, Common Table Expressions (CTEs), and window functions to answer real-world business questions.

---

## 🎯 Objectives

- Analyze pizza sales performance
- Calculate total revenue and order statistics
- Identify top-selling pizzas and categories
- Understand customer ordering patterns
- Analyze sales trends over time
- Generate business insights using SQL

---

## 🛠️ Tools & Technologies

- MySQL
- SQL
- MySQL Workbench

---

## 📂 Database Schema

The project uses the following tables:

- **orders** – Order date and time information
- **order_details** – Quantity of pizzas ordered
- **pizzas** – Pizza size, price, and type
- **pizza_types** – Pizza name and category

---

## 📊 Business Questions Solved

- Retrieve the total number of orders placed
- Calculate total revenue generated from pizza sales
- Identify the highest-priced pizza
- Determine the most commonly ordered pizza size
- List the top 5 most ordered pizza types
- Calculate total quantity sold for each pizza category
- Analyze hourly order distribution
- Find category-wise pizza distribution
- Calculate the average number of pizzas ordered per day
- Determine the top 3 pizza types by revenue
- Calculate each category's percentage contribution to total revenue
- Analyze cumulative revenue over time
- Identify the top 3 highest revenue-generating pizzas within each category

---

## 💡 SQL Concepts Demonstrated

- SELECT Statements
- Filtering & Sorting
- INNER JOIN
- Aggregate Functions (`SUM`, `COUNT`, `AVG`)
- GROUP BY
- ORDER BY
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions (`RANK()`, `OVER()`)
- Date & Time Functions
- Revenue Calculations
